# allow-domains

Репозиторий содержит список доменов, предназначенных для блокировки. Описание репозитория: "List of blocked domains".

## Сырой (raw) файл со списком доменов
Скачать/использовать raw-версию файла:
https://raw.githubusercontent.com/Pixmint/allow-domains/refs/heads/main/all-list.lst

Пример загрузки через curl:

```
curl -L -o all-list.lst "https://raw.githubusercontent.com/Pixmint/allow-domains/refs/heads/main/all-list.lst"
```

## Формат файла
- Каждая строка содержит один домен (например, example.com).
- Пустые строки игнорируются.
- Строки, начинающиеся с `#`, считаются комментариями и игнорируются.

## Применение
Этот список можно использовать в:
- блокировщиках рекламы;
- настройках DNS/hosts;
- брандмауэрах и системах фильтрации трафика;
- скриптах и автоматизированных процессах для обновления правил блокировки.

Пример использования: импортировать `all-list.lst` в файл hosts или в конфигурацию DNS-блокировщика.

## Обновление и вклад
Если вы хотите предложить изменения в списке:
1. Форкните репозиторий.
2. Внесите изменения в файл `all-list.lst` (или создайте новый PR).
3. Откройте Pull Request с описанием изменений и обоснованием.

Пожалуйста, указывайте источник и причину для каждой новой записи в списке.
